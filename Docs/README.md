# Docs/

기획·기술 문서 전체를 관리하는 폴더입니다.

## 구조

| 폴더 | 내용 |
|------|------|
| `GDD/` | 게임 디자인 문서, 시스템 기획서 |
| `TDD/` | 기술 설계 문서, 데이터 구조 정의 |
| `References/` | 레퍼런스 이미지, 무드보드 |
| `MeetingNotes/` | 회의록 |

## 파일 명명 규칙

```
시스템기획서_v1.0.html
회의록_20260327.md
```

## 규칙

- 문서는 `.md` 또는 `.html` 형식을 사용합니다
- 버전 또는 날짜를 파일명에 포함합니다
- 이 폴더는 일반 Git으로 관리됩니다 (LFS 제외)
